CodeGraph Integration

Depends on: eval-cli

Provide LLM agents with deterministic code structure data through apltk CLI. Replaces manual grep/Read discovery with tree-sitter parsed knowledge graph queries.

Submodules

  • cg-lifecycle Service

    Manage CodeGraph index lifecycle — init, sync, status, search

  • cg-discovery Service

    Explore and survey code structure — explore, survey, list-apis

  • cg-validation Service

    Verify architecture proposals and batch-apply atlas mutations

Intra-feature edges

  • cg-lifecycle → cg-discoverydata-rowsurvey/list-apis 依賴已索引的資料庫
  • cg-discovery → cg-validationdata-rowverify 使用 discovery 的搜尋能力
  • cg-validation → cg-lifecycledata-rowapply 非索引操作，不需 lifecycle — 但 template 可搭配 list-apis
