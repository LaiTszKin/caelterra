測試執行器與評分器

並行執行 100 題測試，記錄 JSONL trace，由 LLM-as-Judge 進行多維度評分

Submodules

  • env-utils Service

    .env 環境變數讀取與驗證

  • run-evals Service

    並行測試執行器：隔離環境建立、API 調用、JSONL trace 寫入

  • score Service

    LLM-as-Judge 評分器：trace 解析、多維度評分、問題識別

Intra-feature edges

  • run-evals → env-utilscall讀取 EXEC_*
  • score → env-utilscall讀取 JUDGE_*
  • run-evals → scorecall.done marker 觸發評分
