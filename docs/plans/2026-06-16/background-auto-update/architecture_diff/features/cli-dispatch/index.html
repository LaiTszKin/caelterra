CLI Dispatch and Install Engine

CLI entry point providing argument parsing, command dispatch, and install/uninstall orchestration engine

Submodules

  • arg-parser Service

    parseArguments: CLI argument parsing and dispatch

  • help-builder Service

    buildHelpText family: command help text generation

  • installer-core Service

    installLinks/syncToolkitHome/uninstallSkills: installation engine

  • tool-discovery Service

    registerAllTools/isKnownToolName: dynamic tool loading

  • update-checker Service

    checkForPackageUpdate/execCommand: npm version check

  • background-update-controller Service

    CLI command surface for enabling, disabling, querying, and manually invoking background skill updates

  • background-task-scheduler Service

    Cross-platform registration and removal of daily OS scheduled tasks for Apollo Toolkit updates

  • skill-update-runner Service

    One-shot background update execution that refreshes Apollo Toolkit managed skills and records status
