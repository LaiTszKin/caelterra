# Worker Prompt: T2.1-install-devdeps

- **Source task**: Batch 2 — Install all new devDependencies and add npm scripts

---

## 1. Mission & Rules

### Mission

Install ESLint, typescript-eslint, Prettier, husky, and lint-staged as devDependencies, and add the corresponding scripts to root `package.json`. After installation, run `pnpm exec husky init` to initialize the `.husky/` directory.

### Rules

- Only modify `package.json` (devDependencies and scripts fields)
- Do not change any existing dependency versions
- Do not touch any other files
- After installation, always run `pnpm exec husky init` so the `.husky/` directory is created

---

## 2. Context

### Input Files

- `package.json` — Current root package metadata with dependencies and scripts

### Background

The project needs these quality gate tools added to `devDependencies`:

- `eslint` (v9+) — Linter engine
- `@eslint/js` — ESLint recommended JS rules
- `typescript-eslint` — TypeScript ESLint config + parser (v8, all-in-one package replacing separate @typescript-eslint/parser + @typescript-eslint/eslint-plugin)
- `prettier` (v3+) — Code formatter
- `husky` (v9+) — Git hook manager
- `lint-staged` (v15+) — Staged-file linter runner

New scripts to add:

- `"lint": "eslint . --cache"` — Run ESLint with caching
- `"lint:fix": "eslint . --fix --cache"` — Run ESLint with autofix
- `"format": "prettier --write ."` — Format all files
- `"format:check": "prettier --check ."` — Check formatting without writing
- `"lint-staged": "lint-staged"` — lint-staged CLI
- `"prepare": "husky"` — Required by husky v9 for git hook installation

---

## 3. Tasks

### Task A — Update `package.json` devDependencies

Add the following to the existing `"devDependencies"` object (preserving existing entries):

```json
    "eslint": "^9.0.0",
    "@eslint/js": "^9.0.0",
    "typescript-eslint": "^8.0.0",
    "prettier": "^3.0.0",
    "husky": "^9.0.0",
    "lint-staged": "^15.0.0"
```

### Task B — Update `package.json` scripts

Add the following to the existing `"scripts"` object (preserving `build`, `postbuild`, `prepublishOnly`, `watch`, `test`, `test:coverage`, `test:ts`):

```json
    "lint": "eslint . --cache",
    "lint:fix": "eslint . --fix --cache",
    "format": "prettier --write .",
    "format:check": "prettier --check .",
    "lint-staged": "lint-staged",
    "prepare": "husky"
```

### Task C — Install dependencies

Run: `pnpm install`
Expected: all new devDependencies installed with no errors.

### Task D — Initialize husky

Run: `pnpm exec husky init`
Expected: `.husky/` directory created with a `pre-commit` hook file.

### Output

When done, report back:

- **Files modified**: `package.json` (devDependencies + scripts)
- **Change summary**: 6 devDependencies added, 6 scripts added, husky initialized
- **Test results**: `pnpm install` completed without errors, `.husky/` directory exists
- **Risks or concerns**: None
