## 2. Context

### Input Files

Find all tsconfig.json files in the repository.

### Background

The project currently has `strict: true` in all tsconfigs. The three additional flags provide extra type safety:

- `noUncheckedIndexedAccess`: Adds `undefined` to every indexed access type (`arr[i]` is `T | undefined`)
- `exactOptionalPropertyTypes`: Distinguishes `prop?: T` (may be absent) from `prop: T | undefined` (always present, may be undefined)
- `noPropertyAccessFromIndexSignature`: Requires bracket notation when accessing properties via an index signature

These flags were confirmed in the quality-gate-setup SPEC.md and DESIGN.md.
