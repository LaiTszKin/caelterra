## 1. Mission & Rules

### Mission

Fix all ESLint `strict-type-checked` violations and TypeScript strict flag violations in the remaining tool packages: architecture, create-review-report, create-specs, find-github-issues, open-github-issue, read-github-issue, review-threads, validate-openai-agent-config, validate-skill-frontmatter.

### Rules

- Only modify `.ts` source files in the allowed directories
- Do not change any runtime logic or test assertions
- Prefer `unknown` over `any`
- Only use `eslint-disable` for genuine false positives
