## 1. Mission & Rules

### Mission

Create three configuration files: `eslint.config.mjs` (ESLint flat config with strict-type-checked), `.prettierrc` (Prettier config with singleQuote), and `.lintstagedrc.json` (lint-staged file pattern mapping).

### Rules

- Create all three files at the repository root
- Do not modify any existing files
- Do not run any commands (config files only)

---

## 2. Context

### Background

The project uses ESLint v9+ flat config, typescript-eslint v8, Prettier v3, and lint-staged v15.

**ESLint config requirements** (from design research):

- Flat config (`eslint.config.mjs`)
- Import from `@eslint/js` and `typescript-eslint`
- Use `tseslint.configs.strictTypeChecked` (the `strict-type-checked` config array)
- Use `parserOptions.projectService: true` for monorepo type-aware linting (v8 recommended approach)
- Use `tsconfigRootDir: import.meta.dirname` for path resolution
- Use `--cache` in the lint script for performance (set up in scripts, not config)

**Prettier config requirements**:

- `singleQuote: true`
- All other options left as Prettier defaults

**lint-staged config requirements**:

- Run ESLint fix + Prettier on staged TypeScript files
- Run Prettier on staged JSON files (if any)
- Run Prettier on staged JS/MJS files

## 3. Tasks

### Task A — Create `eslint.config.mjs`

Create the file at the repository root:

```js
// @ts-check

import eslint from '@eslint/js';
import tseslint from 'typescript-eslint';

export default tseslint.config(
  eslint.configs.recommended,
  ...tseslint.configs.recommendedTypeChecked,
  ...tseslint.configs.strictTypeChecked,
  {
    languageOptions: {
      parserOptions: {
        projectService: true,
        tsconfigRootDir: import.meta.dirname,
      },
    },
  },
  {
    // Allow test files to use any types more freely
    files: ['**/*.test.ts', '**/*.test.js'],
    rules: {
      '@typescript-eslint/no-unsafe-assignment': 'off',
      '@typescript-eslint/no-unsafe-member-access': 'off',
      '@typescript-eslint/no-unsafe-call': 'off',
      '@typescript-eslint/no-unsafe-return': 'off',
    },
  },
  {
    ignores: [
      'dist/',
      'node_modules/',
      '.claude/',
      '.codegraph/',
      '.apollo-toolkit/',
      'resources/',
      'docs/plans/**/architecture_diff/',
      'skills/init-project-html/sample-demo/',
      '*.test.js',
      'test/**/*.test.js',
    ],
  },
);
```

### Task B — Create `.prettierrc`

```json
{
  "singleQuote": true
}
```

### Task C — Create `.lintstagedrc.json`

```json
{
  "*.{ts,mjs,js,cjs}": ["eslint --fix", "prettier --write"],
  "*.json": ["prettier --write"],
  "*.{yaml,yml}": ["prettier --write"],
  "*.md": ["prettier --write"]
}
```
