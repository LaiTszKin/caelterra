## 1. Mission & Rules

### Mission

Add the minimal external dependency needed for package extraction, refresh the lockfile, and run the final verification suite for the background auto-update feature.

### Rules

- Follow the Scope in Section 5 — only modify files listed as Allowed.
- Do not change feature code logic unless a dependency import path needs a trivial adjustment.
- Do not widen the dependency set beyond what the runner actually uses.
- Workers are leaf nodes — do not spawn sub-workers.

### `package.json` and `package-lock.json` — add `pacote`

1. Open `package.json`.
2. Add `pacote` as a normal dependency using the smallest compatible semver range consistent with the design.
3. Update `package-lock.json` by running the package manager install/update command used by this repo.

### Final verification

1. Run the full build and test suite.
2. Run the focused feature tests added by earlier workers.
3. Record any remaining failures as coordinator-blocking issues instead of patching unrelated files.
